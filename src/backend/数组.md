# 数组

## 一、数组的特点

* 数组就是一组数，一组相同类型元素的集合
* 最常用一维数组，二维数组
* 元素类型必须一致（如全是int、全是string）;
* 元素在内存中连续存储，通过索引（从0开始）访问；
* 长度固定（一旦创建，不能动态扩容，需通过其他方式间接实现扩容）；

## 二、一维数组的声明与初始化

* 数组的使用分为“声明”和“初始化”两步：
  1. 声明数组
     * 语法：数据类型[] 数组名；
     * `int[] scores; //声明一个int类型的数组`
     * `string[] name; //声明一个string类型的数组`
  
  2. 初始化数组
     * 初始化需指定数组长度或直接提供元素，有3种常用方式：
         1. 直接赋值初始化（最简洁）
         无需指定长度，编译器自动根据元素数量决定长度：
         `int[] scores = {90,85,95,78}; //长度为4的int数组`
         `string[] name = {"张三","李四","王五"}； //长度为3的string数组`

         2. 指定长度初始化（元素为默认值）
         通过 new 关键字指定长度，元素会被自动赋予默认值（如int默认0，string默认null）：
         `int[] nums = =new int[3]; //长度为3的int数组，元素默认0，0，0`
         `string[] strs = new string[2]; //长度为2的string数组，元素默认值null`

         3. 结合 new 和元素初始化
         显示用 new 指定类型和长度，同时赋值（长度必须与元素数量一致）：
         `int[] ages = new int[3] {10,20,30}; //长度3，元素数量3（正确）`
         `// int[] ages = new int[2] {10,20,30}; //错误，长度2与元素数量3不匹配`

## 三、数组的访问与修改

* 通过索引（从 0 开始）访问或修改元素，格式：数组名[索引]
* `int[] scores = {90,85,95};`
* `//访问元素：`
* `Console.WriteLine(scores[0]); //输出：90（第一个元素）`
* `Console.WriteLine(scores[2]); // 输出：95（第三个元素）`
* `//修改元素：`
* `scores[1] = 100;`
* `Console.WriteLine(scores[1]); // 输出：100（第二个元素被修改）`

   **⚠️ 注意：索引不能超出范围（0 ≤ 索引 < 数组长度），否则会抛出IndexOutOfRangeException异常**

## 四、数组的长度（Length属性）

每个数组都有Length属性，用于获取元素的总数量：
int[] nums = {1,2,3,4};
Console.WriteLine(nums.Length); //输出：4（数组长度为4）

## 五、多维数组

C# 支持多维数组（如二维、三维），最常用的是二维数组

1. **二维数组（矩阵形式）**
声明语法：数据类型[,] 数组名；（注意用逗号分隔维度）
初始化方式：
`//方式1：直接赋值（2行3列）`
`int[,] matrix = {`
    `{1,2,3},`
    `{4,5,6}`
`};`

`//方式2：指定维度长度`
`int[,] matrix2 = new int[2,3];`

`//方式3：结合new和元素赋值`
`int[,] matrix3 = new int[2,3]{`
    `{1,2,3},`
    `{4,5,6}`
`};`

访问元素：通过[行索引，列索引] (均从0开始)：
`Console.WriteLine(matri[0,1]);`
`matri[0,1] = 100; //修改第一行第二列元素为100`

2. **交错数组（数组的数组）**

与二位数组不同，交错数组的“子数组”长度可以不同（更灵活）
声明语法：数据类型[][] 数组名；
初始化方式：
//步骤一：声明交错数组（先指定“外层数组”长度）
int[][] jagged = new int[2] [];

//步骤二：初始化每个子数组（长度可不同）
jagged[0] = new int[] {1,2};//第0个子数组长度为2
jagged[1] = new int[] {3，4，5}； //第一个子数组长度为3

## 六、数组的常用操作

通过Systme.Array 类提供静态方法，可以便捷操作数组：

1. **遍历数组**
   * for循环：适合需要索引的场景；
   * foreach循环：适合仅遍历元素（无需索引）
   * `int[] nums = {1,2,3,4};`
   
   * `//for 循环遍历`
   * `for (int i = 0;i < nums.Length;i++)`
   * `{`
   * `Console.WriteLine(nums[i]);`
   * `}`
   
   * `//foreach循环遍历（更简洁）`
   * `foreach (int num in nums)`
   * `{`
   * `Console.WriteLine(num);`
   * `}`

2. **数组排序（Sort）**
   Array.Sort(数组)：对数组元素升序排序（默认）：
   int[] nums = {3,1,4,2};
   Array.Sort(nums); //排序后1，2，3，4

3. **数组反转（Reverse）**
   Array.Reverse(数组)：反转数组元素顺序：
   int[] nums = {1,2,3};
   Array.Reverse(nums); //反转后3，2，1

4. **数组复制（Copy）**
   Array.Copy(源数组，目标数组，复制长度)：将源数组的指定长度元素复制到目标数组：
   `int[] source = {1,2,3};`
   `int[] target = new int[3];`
   `Array.Copy(source,target,3); //复制后target：{1，2，3}`

## 七、注意事项

1. 默认值：未赋值的数组元素会有默认值（值类型为int值为0，引用类型为string值为null）；
2. 引用类型特性：数组是引用类型，赋值时传递的是内存地址（修改副本会影响元数组）：
   int[] a = {1,2};
   int[] b = a; //b和a指向同一个数组
   b[0] = 100;
   console.WriteLine(a[0]); //输出：100（a被b修改）

3. 边界检查：访问索引超出范围会抛出Index0ut0fRangeException,需避免
