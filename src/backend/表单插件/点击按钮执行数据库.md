# 点击按钮执行数据库，更新全部单据

执行SQL，给单据批量赋值

```示例
//点击按钮执行SQL数据
//更新全部单据备注信息
public override void BarItemClick(BarItemClickEventArgs e)
{
    base.BarItemClick(e);
    //当点击OXCH_tbTest按钮时触发
    if(e.BarItemKey.EqualsIgnoreCase("OXCH_tbTest"))
    {
        //执行sql语句返回int，表示影响了多少行
        int x = DBUtils.Execute(this.Context, "/*dialect*/update T_PUR_POORDERENTRY set FNote = '测试'");

        //弹窗提示
        this.View.ShowMessage(x.ToString());
    }
}
```
