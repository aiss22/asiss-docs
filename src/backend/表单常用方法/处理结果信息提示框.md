# 处理结果信息提示框

//处理结果信息提示框

```示例
IOperationResult opResult = new OperationResult();//引用变量
//实例化
opResult.OperateResult.Add(new OperateResult()
{
    Name = "信息提示",
    //成功状态提示
    Message = "成功",
    SuccessStatus = true

});
//显示处理结果
this.View.ShowOperateResult(opResult.OperateResult);

opResult.OperateResult.Add(new OperateResult()
{
    Name = "信息提示",
    Message = "失败",
    SuccessStatus = false
});
this.View.ShowOperateResult(opResult.OperateResult);
```

## 处理结果信息提示框 循环

业务场景：用于批处理任务，告诉用户，哪些处理成功，哪些处理失败；
//处理结果信息提示框 循环

```示例
IOperationResult opResult = new OperationResult();
for(int i = 1; i < 20; i++)
{
    //如果为奇数就显示成功
    if(Convert.ToBoolean(i%2))
    {
        opResult.OperateResult.Add(new OperateResult()
        {
            Name = "信息提示",
            Message = i.ToString()+"---奇数成功",
            SuccessStatus = true
        });
    }
    //否则显示失败
    else
    {
        opResult.OperateResult.Add(new OperateResult()
        {
            Name = "信息提示",
            Message = i.ToString()+"---偶数失败",
            SuccessStatus = false
        });
    }
    
}
this.View.ShowOperateResult(opResult.OperateResult);
```
