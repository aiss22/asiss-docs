# 表单常用方法：获取单据标题

`this.View.GetFormTitle();` 获取单据标题

```获取单据标题
public override void BarItemClick(BarItemClickEventArgs e)
{
    base.BarItemClick(e);//重写按钮点击事件
    if(e.BarItemKey.EqualsIgnoreCase("OXCH_tbHuoQu"))//判断按钮
    {
        string FormTitle = this.View.GetFormTitle();
        this.View.Model.SetValue("FNote",FormTitle);//赋值显示
        this.View.ShowMessage("单据标题为："+FormTitle);//弹消息提示
    }
}

```
