# 隐藏、锁定按钮

`this.View.GetMainBarItem("按钮标识").Enabled = false;` 锁定按钮

```示例
//点按钮锁定单据头保存字段
this.View.GetMainBarItem("tbSplitSave").Enabled = false;
this.View.GetMainBarItem("tbSave").Enabled=false;
```

`this.View.GetMainBarItem("按钮标识").Visible = false;` 隐藏按钮

```示例
//点按钮隐藏单据头保存字段
this.View.GetMainBarItem("tbSplitSave").Visible = false;
```
