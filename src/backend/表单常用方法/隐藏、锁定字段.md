# 隐藏或锁定字段

`this.View.GetControl("字段标识").Enabled = false;` 锁定单据头字段

```示例
//点击按钮，备注字段锁定，变成灰色
this.View.GetControl("FNote").Enabled = false;
```

`this.View.GetFieldEditor("单据体（分录行）字段标识",0).Enabled=false;` 锁定单据体分录字段

```示例
//点击按钮，单据体（分录行）备注变灰
this.View.GetFieldEditor("FEntryNote",0).Enabled=false;
```

`this.View.GetControl("字段标识").Visible = false;` 隐藏字段

```示例
//点按钮隐藏备注字段
this.View.GetControl("FNote").Visible = false;
```
