# 异常处理

* **异常处理：用 try-catch-finally 捕获错误，避免程序崩溃**

```string s="abc";
try
{
    int a=int.Parse(s);
    Console.WriteLine(a);
}
catch (FormatException ex)//具体的
{
    Console.WriteLine($"输入的不是数字：{ex.Message}");
}

catch (Exception ex) //通用的，写在最后兜底
  {
   Console.WriteLine("出错了："+ ex.Message);
  }

finally //无论是否出错都执行（如释放资源）
  {
   Console.WriteLine("处理结束")；
  }
```

**枚举（enum）:定义命名常量集合，增强可续性**
  `enum Season { Spring, Summer, Autumn, Winter}`
  `Season s = Season.Summer; //使用枚举值`
