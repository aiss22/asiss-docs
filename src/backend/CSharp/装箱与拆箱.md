# 装箱与拆箱

装箱将值类型存储在堆上（作为对象），拆箱提取值到栈
值类型与引用类型之间转换：

```class Program
{
    static void Main()
    {
        int value = 42;            // 值类型
        object boxed = value;      // 隐式装箱为 object

        Console.WriteLine("装箱后: " + boxed);

        int unboxed = (int)boxed;  // 显式拆箱回 int

        Console.WriteLine("拆箱后: " + unboxed);
    }
}
```
