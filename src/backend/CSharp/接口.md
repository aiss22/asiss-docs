# Interface 接口

在 C#中实现抽象的另一种方式是通过接口
与抽象类类似，接口不能用于创建对象
实现接口时必须实现接口所有方法
接口不能包含构造函数

```接口
// Interface 接口
interface IAnimal 
{
  void animalSound(); // 接口方法（没有方法体）
}

// Pig “实现” 了 IAnimal 接口
class Pig : IAnimal 
{
  public void animalSound() 
  {
    // 接口方法 animalSound () 的方法体在此处提供
    Console.WriteLine("The pig says: wee wee");
  }
}

class Program 
{
  static void Main(string[] args) 
  {
    Pig myPig = new Pig();  // 创建一个 Pig 对象
    myPig.animalSound();
  }
}
```
