# Html & Css

`<xxx>hello</xxx>` 标签

`<a href="链接" target="_blank">点击进入</a>` 超链接,并且在新页面打开

属性的语法规则：属性和标签名中间空格隔开，后加 = ""
`href=""` 属性:链接路径
`target="_blank"`属性：新页面打开链接
(这两个属性只对链接元素有效)

## Css

`background-color`背景颜色
`color`颜色
`border`边缘
`border: none;`取消边缘线
`cursor: pointer;`鼠标样式
`margin`边缘空白
`padding`填充
`transition`过度（例如鼠标触碰按钮柔和过度0.15s）
过度（多个元素）：`transition:background-color 0.15s,color 0.15s;`
`opacity`不透明度
`box-shadow`阴影
示例：`box-shadow: 5px 5px 10px rgba(0,0,0,0.15);`

`.p`css选择器
`:hover`鼠标悬停
`:active`点击
